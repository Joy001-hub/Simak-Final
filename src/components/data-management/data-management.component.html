Panduan Lengkap Manajemen Data

1. Bagaimana Data Anda Disimpan?

Untuk menjaga privasi dan memberikan Anda kontrol penuh, semua data aplikasi ini (penjualan, kavling, pelanggan, dll.) disimpan dengan aman di Penyimpanan Lokal (Local Storage) browser Anda. Data tidak dikirim atau disimpan di server online manapun.

Artinya, data TIDAK AKAN HILANG jika Anda me-refresh halaman, menutup browser, atau mematikan komputer.

2. PENTING: Risiko Kehilangan Data

Karena data disimpan di browser, data akan HILANG SECARA PERMANEN jika Cache/Data Browser terhapus, baik secara sengaja maupun tidak sengaja.

  • Contoh sengaja: Anda menggunakan fitur "Clear browsing data" di Chrome/Firefox dan mencentang opsi "Cookies and other site data" atau "Cached images and files".
  • Contoh tidak sengaja: Aplikasi pembersih (seperti CCleaner) berjalan otomatis dan membersihkan data browser, atau Anda melakukan install ulang/reset browser.

Solusi: Selalu backup data Anda menggunakan fitur "Export Data (JSON)" secara rutin.

3. Penjelasan Fitur-Fitur

  • Muat Ulang Data Contoh: Mengisi aplikasi dengan data demo untuk keperluan percobaan atau belajar. Perhatian: Tindakan ini akan menghapus semua data yang ada saat ini.
  • Export Data (JSON): Membuat file backup (.json) dari seluruh data Anda. Ini adalah 'brankas digital' Anda. Sangat disarankan untuk melakukan ini secara rutin, terutama setelah menambahkan banyak data baru.
  • Import Data (JSON): Memulihkan data dari file backup yang sebelumnya Anda simpan. Gunakan fitur ini saat Anda berpindah perangkat atau jika data tidak sengaja terhapus. Perhatian: Proses ini akan menimpa (overwrite) semua data yang ada di aplikasi dengan data dari file backup.
  • Reset Semua Data Aplikasi: MENGHAPUS SEMUA DATA SECARA PERMANEN dan mengembalikan aplikasi ke kondisi awal (kosong). Gunakan fitur ini hanya jika Anda benar-benar ingin memulai dari awal. Tindakan ini tidak dapat diurungkan.
